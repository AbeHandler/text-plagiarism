This directory contains three sub-directories and two files:

1) annotated: This contains a pilot sample of 25 texts marked up manually via EMACS.

2) rawtexts:  This contains all original newspaper reports scanned in.

3) originals_backup:  This contains back-up files.

4) File "court_whollynonderived_index.txt" keeps an index of newspaper articles which are not derived from PA, but are about the same topics as PA copies of the same catchlines. File "nonderived_files.txt" keep their file locations in the METER Corpus.

5) File "court_whollyderived_index.txt" keeps an index of newspaper articles which are wholly derived from PA.
